# Pré-visualização da interface

Para **ver só o layout** (cores, tipografia, telas) **sem instalar Node nem MySQL**:

1. Baixe ou clone o repositório.
2. Abra no navegador o arquivo:

   **`docs/previews/interface-demo.html`**

   *(Duplo clique no arquivo ou arraste para o Chrome / Edge / Firefox.)*

Esse HTML é **estático**: dados são fictícios. O funcionamento real está em `frontend/index.html` com a API em `backend/` (veja o `README.md` na raiz do projeto).
